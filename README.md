# NEX Engine

NEX Engine is the next-generation game engine designed to combine the best of both worlds from Blender and Unreal Engine.

## Project Goals
- Simplify 3D modeling, sculpting, and animation.
- Provide advanced yet easy-to-use game mechanics.
- Ensure high performance across platforms.

## How to Contribute
Feel free to contribute by creating issues or submitting PRs to add features.
